# ShotLabs

**ShotLabs** is an open-source initiative by **Project 89** to democratize VRM avatar creation and animation.

We provide tools, pipelines, and experiments that make it easier for developers, creators, and communities to build, customize, and animate VRM characters for games, virtual worlds, streaming, and AI-driven experiences.
